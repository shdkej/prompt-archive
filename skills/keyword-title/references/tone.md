## 3. 매체별 톤 분기

### 블로그 제목 (shdkej.com)
- **결론형**: "X로 Y한 이야기" — 사건+감상 (예: "Slack 배포봇으로 배포시간 70% 줄인 이야기")
- **이야기형**: "X에서 막힌 날" — 시간순 구성 암시 (예: "EKS 1.20→1.27 마이그레이션에서 막힌 날")
- **선택의 순간형**: "X vs Y, 이 상황에서는 Z를 골랐다" (예: "Lambda vs ECS, 분당 5만건에서는 ECS를 골랐다")
- 길이: 25~45자
- 숫자/고유명사 우선

### Threads 제목 (첫 줄)
- 250자 본문의 **첫 한 줄**로 시선을 잡는 역할
- 후크가 있되 자극적이지 않아야 함
- 길이: 15~30자
- 톤: 회고형 or 발견형
- 예시:
  - "오늘은 기술보다 순서가 중요하다는 걸 다시 배웠습니다."
  - "3월 한 달 동안 회사에서 AX 업무를 하면서 여러 자동화를 직접 붙여봤다."
  - "큰 거 하나보다 작은 거 몇 개 정리한 게 더 잘한 일이었다."

### YouTube 제목 (예외 분기 — 후크 허용)
> 유튜브는 위 블로그·Threads와 **톤 기준이 다르다**. 시청자가 주어이고, 클릭이 목적이다.
> §2의 절대규칙 중 **3번(자극적 후크)·6번(단정 회피)은 완화**한다. 단 **거짓 낚시는 여전히 금지**.

- 강한 후크 허용: 호기심 갭, 큰 숫자 대비, 결과 숨기기
- 결론을 일부러 감춰도 됨 ("그래서 어떻게 됐을까")
- 길이: 20~40자 (모바일에서 잘리지 않는 선)
- **여전히 금지**: 영상에 없는 내용 약속(거짓 낚시), 공포 마케팅("이거 모르면 망함"), 형용사 남발("역대급·충격적")
- 검색 키워드(Searchability)를 반드시 1개 이상 포함
- 통과 예시:
  - "회사 그만두고 신혼여행 1년, 통장 잔고 공개합니다"
  - "1년 세계여행 끝나고 우리가 달라진 것"
- 폐기 예시:
  - "충격! 신혼여행 갔다가 인생 망한 썰" (공포·과장)
  - "이거 모르면 세계여행 가지 마세요" (공포 마케팅)

## 4. THREADS_STYLE_HISTORY 누적 합의 (2026-03-13 기준)

- "좀 더 라이트" — 무겁지 않게
- "약간 느끼해" — 미사여구 줄이기
- 3~5줄 안에 핵심 전달
- 줄글 형태 선호, 정형 블록 포맷 지양
- 시리즈형 글은 **메인 + 첫 댓글 직접 달기** 구조 (운영 메모 2026-04-01)
- 본문 말미에 **인용/리포스트되기 쉬운 한 줄**

## 5. 자가 체크리스트 (제목 후보별)

각 제목 후보를 출력하기 전, 아래 6개 항목을 통과해야 한다.

```
[ ] 격식체 금지 — "~하십시오" 없음
[ ] 자극적 후크 금지 — "충격", "이거 모르면" 등 없음
[ ] AI 생성티 금지 — "안녕하세요", "오늘은 ~에 대해" 없음
[ ] 1인칭 경험 포함 — "내가", "겪었다", 구체 수치/고유명사 중 하나 이상
[ ] 길이 적정 — 블로그 25~45자, Threads 15~30자
[ ] 단정 회피 — "정답"이 아니라 "선택"의 톤
```

체크 6/6: 출력 / 5/6 이하: 재생성 또는 폐기

### YouTube 모드 자가체크 (위 6항목 대신 적용)

```
[ ] 거짓 낚시 금지 — 영상에 없는 내용 약속 안 함
[ ] 공포 마케팅 금지 — "이거 모르면", "충격" 등 없음
[ ] 형용사 남발 금지 — "역대급·충격적" 등 과장 수식 없음
[ ] 검색 키워드 포함 — Searchability 명사구 1개 이상
[ ] 호기심 갭 존재 — 결과를 다 말하지 않음
[ ] 길이 적정 — 20~40자
```

체크 6/6: 출력 / 5/6 이하: 재생성 또는 폐기
