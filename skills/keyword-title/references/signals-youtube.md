# YouTube Title Signals (유튜브 제목·썸네일 카피 맥락)

> 콘텐츠 모드(blog·threads)의 `signals-content.md`와 **다른 평가축**을 쓴다.
> 블로그는 "내가 겪은 이야기"(PersonalDepth 0.30 최상위)가 핵심이지만,
> 유튜브 제목은 **"시청자가 아직 못 본 결과·변화"**가 주인공이다.
> 시청자가 주어다. 내 경험은 후크의 재료일 뿐, 가중치가 아니다.
>
> MVP 원칙은 동일: 외부 API 없이 LLM 추론만으로 평가. (실측은 사용자가 vidIQ·네이버 데이터랩 등으로)

---

## 1. 신호 목록 (6종)

| 신호 | 의미 | 측정 (LLM 추론) | 가중치 |
|---|---|---|---|
| **CuriosityGap** | 결과를 숨겨 "그래서 어떻게 됐는데?"를 만든다 | 제목이 결론을 끝까지 감추는가 | **0.25** |
| **Searchability** | 사람들이 실제 검색창에 치는 키워드를 포함 | 핵심 명사구가 검색 의도와 일치하는가 | **0.20** |
| **NumberContrast** | 구체 숫자 + before/after 대비 폭 | 숫자·기간·금액·배수가 있고 대비가 큰가 | 0.15 |
| **StakesChange** | 판돈·리스크·극적 변화 | "그만두고", "전 재산", "0원→" 같은 전환점 | 0.15 |
| **Relatability** | "나도 저럴 수 있다" 공감 진입점 | 시청자가 자기 상황에 대입 가능한가 | 0.15 |
| **Authenticity** | 영상 내용을 배신하지 않는 정직한 후크 | 과장·거짓 약속이면 **감점** (clickbait 방지) | 0.10 |

> **CuriosityGap을 최상위(0.25)로 두는 이유**: 유튜브는 검색(Searchability)으로 노출되고 **호기심 갭으로 클릭**된다. 결과를 다 말하는 제목("1년 세계여행 다녀온 이야기")은 클릭할 이유가 없다.
> **Authenticity가 음의 안전장치**: 갭을 키우려고 영상에 없는 걸 약속하면 시청 유지율이 박살나고 알고리즘이 처벌한다. 그래서 거짓 낚시는 가중치가 아니라 **감점**으로 작동한다.

---

## 2. 점수 산식

```
YT Score (0~100) =
  100 × (
    0.25 · CuriosityGap
  + 0.20 · Searchability
  + 0.15 · NumberContrast
  + 0.15 · StakesChange
  + 0.15 · Relatability
  + 0.10 · Authenticity
  )

각 신호는 0.0 ~ 1.0
```

### 컷오프
- **70점 이상**: 메인 후보 (제목으로 바로 써볼 만함)
- **55~69점**: A/B 테스트용 (썸네일 카피와 조합해 반응 보고 결정)
- **55점 미만**: 출력 목록에서 제외 (`--include-low`로 강제 시 표시)

> 블로그(70/50/<50)보다 중간 구간을 55로 올린다. 유튜브는 "그럭저럭"이 가장 위험하다 — 클릭이 안 되면 데이터 자체가 안 쌓인다.

## 3. 각 신호의 LLM 평가 가이드

### CuriosityGap
- 제목을 읽고도 "그래서 결과가 뭔데?"가 남는가? → 1.0
- 결과를 일부만 흘림 (숫자는 주되 어떻게는 숨김): 0.6
- 결론을 다 말해버림 ("~다녀온 이야기"): 0.2

### Searchability
- 사람들이 검색창에 칠 핵심 명사구가 그대로 들어있는가? (예: "세계여행", "신혼여행 비용") → 1.0
- 검색어가 비유·은유에 묻혀 있음: 0.5
- 검색 의도와 무관한 감성 문장만: 0.2

### NumberContrast
- 구체 숫자 + 큰 대비가 둘 다 있음 ("회사 관두고 1년, 통장 0원에서 시작") → 1.0
- 숫자만 있고 대비는 약함 ("1년 세계여행"): 0.6
- 숫자 없음: 0.2

### StakesChange
- 돌이킬 수 없는 결정·큰 리스크·인생 전환점이 보임 → 1.0
- 변화는 있으나 안전한 범위: 0.5
- 일상적, 판돈 없음: 0.2

### Relatability
- 시청자 다수가 "나도 고민해본 것"에 닿음 (퇴사·돈·결혼) → 1.0
- 일부 공감: 0.5
- 특정 소수만의 경험: 0.3

### Authenticity (감점 신호)
- 영상이 실제로 보여줄 내용과 제목이 일치함 → 1.0
- 약간 부풀림 (그래도 핵심은 사실): 0.6
- 영상에 없는 걸 약속 / 공포·과장 낚시 ("이거 모르면 망함"): **0.1 이하, 톤 위반으로 폐기 검토**

## 4. blog/threads 신호와의 차이 요약

| 항목 | signals-content (blog·threads) | signals-youtube |
|---|---|---|
| 주어 | 나(내 경험) | 시청자(못 본 결과) |
| 최상위 신호 | PersonalDepth 0.30 | CuriosityGap 0.25 |
| 결론 노출 | 결론형 제목 OK | 결론 숨김이 유리 |
| 후크 강도 | 담백·차분 (자극 금지) | 강한 후크 허용 (거짓만 금지) |
| 컷오프 중간대 | 50~69 | 55~69 |
| Authenticity | 별도 없음(톤이 흡수) | 독립 감점 신호로 분리 |

## 5. 자동 폐기 조건

아래 중 하나라도 걸리면 점수와 무관하게 폐기:

1. 영상에 없는 내용을 약속하는 **거짓 낚시** (Authenticity ≤ 0.1)
2. 공포 마케팅 ("이거 모르면 망합니다", "충격") — tone.md §2-3 위반
3. 검색 키워드가 전혀 없어 노출 경로가 0 (Searchability ≤ 0.2)
4. 같은 채널 기존 영상 제목과 80%+ 중복

## 6. 외부 API 연동 — YouTube Data API (Searchability 1순위) + 데이터랩(보조)

### ★ Searchability는 `scripts/youtube_keyword.py`로 측정한다 (데이터랩 아님)

```
python3 scripts/youtube_keyword.py --keywords "신혼여행,세계여행,세계일주" --region KR --max 10
```

- **교체되는 신호**: `Searchability` ← 키워드 상위 영상의 `median_view`(조회수 중앙값) 기반 0~1.
  - median_view ≥ 100k →1.0 / ≥30k →0.8 / ≥10k →0.6 / ≥3k →0.4 / <3k →0.2
  - `recent_ratio`(최근 1년 발행 비율) 참고: 낮으면 **에버그린**(오래된 영상도 계속 소비), 높으면 신선·경쟁 활발.
- **정성 신호 유지**: CuriosityGap·NumberContrast·StakesChange·Relatability·Authenticity는 LLM 추론.

> **⚠️ 플랫폼 역전 주의 (2026-06-03 실측 발견)**: 네이버 검색량과 유튜브 조회수는 **자주 정반대**다.
> 예) "세계일주"는 네이버 검색량 최저(−57% 급락)지만 유튜브 median 785k로 압도적 1위(에버그린).
> "신혼여행"은 네이버 검색 1위지만 유튜브 조회는 중간.
> → **유튜브 제목의 Searchability는 반드시 YouTube API로 본다. 데이터랩으로 대체하면 키워드를 거꾸로 고른다.**

### 데이터랩은 "주제 트렌드 참고"용 (보조)
- `scripts/datalab_trend.py`의 `velocity_signal`은 "네이버에서 이 주제가 뜨는지" 참고용. 유튜브 점수엔 직접 안 들어간다.

### Fallback
- 둘 다 `ok=false`(키 없음/오류)면 LLM 추론으로 진행. 키 위치·패턴은 `signals-content.md §5`와 동일(`GOG_YOUTUBE_API_KEY`).
- 쿼터 주의: `search.list`=100 units/키워드. 일일 기본 10,000 → 키워드 ~100개/일.
