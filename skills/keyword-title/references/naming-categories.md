# Naming Categories (다양성 강제용)

> 후보를 한 카테고리에 몰리지 않게 강제 분산하는 도구.
> Divergent 단계에서 카테고리당 2~3개씩 생성 → Convergent에서 카테고리 분산을 보장하며 5~7개로 추림.

---

## 1. 10개 카테고리

| # | 카테고리 | 한 줄 정의 | 예시 (분야 무관) |
|---|---|---|---|
| 1 | **의태어/구어체** | 한국어 의태/의성/구어 표현 단독 | 차곡차곡, 톡톡, 잘잘잘, 슬슬 |
| 2 | **직역/직설** | 본질을 그대로 진술 | 다음 생, 오늘의 덕, 일기장 |
| 3 | **메타포** | 본질의 도구/장면을 비유로 | 덕수첩, 거울, 등불, 둥지 |
| 4 | **신조어/조어** | 기존 단어를 합쳐 새 단어 | 오늘덕, 환생도감, 덕폴 |
| 5 | **시적/추상** | 시·문장 톤, 여운 우선 | 이번 생부터, 그러는 사이, 가만히 |
| 6 | **펀/말맛** | 이중 의미, 발견의 재미 | 덕분에 (덕+분), 잘함점, 좋은 척 |
| 7 | **의인화** | 캐릭터/페르소나 명명 | 덕이, 덕이네, 우리 환생씨 |
| 8 | **동사형/명령형** | 행동을 호출하는 톤 | 쌓아둬, 모아둬, 다시 한 번 |
| 9 | **외래어/합성** | 한·영 또는 한·일 합성 | Karma+, RebirthDex, Brush |
| 10 | **한자/문어** | 한자 단어 차용 (단, BRAND 톤 위험) | 功徳, 善業, 餘生 |

## 2. 카테고리별 강점과 함정

### 1. 의태어/구어체 ★
- **강점**: 한국어 사용자에게 가장 따뜻함. BRAND 톤 자연 정합.
- **함정**: 일반명사화 위험 (`차곡차곡`은 너무 자주 쓰이는 단어), 검색 식별성 약함.
- 적합한 제품: 일상 도구, 일기/기록, 명상

### 2. 직역/직설 ◆
- **강점**: 컨셉이 즉시 전달. 카피 자연스럽게 붙음.
- **함정**: 일반명사화 위험 (직설일수록 차별성 ↓). 첫 만남에서 호기심 약함.
- 적합한 제품: 카테고리 1위가 목표인 도구

### 3. 메타포 ★
- **강점**: 한 단어에 컨셉+도구 이미지를 동시 전달. 기억성·확장성 모두 좋음.
- **함정**: 메타포가 너무 멀면 컨셉 불명확. **본질에서 1걸음 거리**가 최적.
- 적합한 제품: 정체성 자체가 메타포인 도구 (수첩·도감·일기·거울 같은 메타 객체)

### 4. 신조어/조어 ◆
- **강점**: 검색 식별성 최상 (구글 1페이지 독점 가능).
- **함정**: 첫 만남에서 의미 전달 약함 → 광고/카피로 의미 주입 필요. 트렌드성 단어와 결합하면 3년 후 촌스러움.
- 적합한 제품: 마케팅 예산 있는 제품

### 5. 시적/추상 △
- **강점**: 여운, 호기심 유발.
- **함정**: 검색·기억성 모두 약함. 사용자 첫 만남에 "이게 뭐 하는 거지" 의문.
- 적합한 제품: 명상·일기·심리 등 추상적 가치 제품

### 6. 펀/말맛 ★
- **강점**: 한 박자 후 의미 풀림 → 말맛이 강하면 입소문 자동.
- **함정**: 위트가 컨셉과 안 맞으면 그냥 유치함. 한국어/영어 펀은 글로벌 확장 시 무력.
- 적합한 제품: 가벼움이 본질인 제품 (게임·소셜·라이프스타일)

### 7. 의인화 ◆
- **강점**: 친근감 최고. UI 카피와 자연 연결 ("덕이가 보고있어요").
- **함정**: 진지한 제품에선 어색. 캐릭터 디자인 의존도 ↑.
- 적합한 제품: 라이프 게임·교육·아이용 제품

### 8. 동사형/명령형 △
- **강점**: 행동 유도. 첫 화면 인터랙션과 정합.
- **함정**: 명령형은 BRAND 톤(부드러운 존댓말)과 살짝 충돌 위험. 음절수 조절 필요.
- 적합한 제품: 액션이 핵심인 제품 (배달·운동·기록)

### 9. 외래어/합성 △
- **강점**: 글로벌 확장 용이. 도메인 가용성 좋음.
- **함정**: BRAND "따뜻함"과 어울리기 어려움. 한국어 사용자에게 거리감.
- 적합한 제품: B2B/개발자 도구/SaaS

### 10. 한자/문어 ✗ (BRAND에서 거의 회피)
- **강점**: 깊이/품격.
- **함정**: 차가움·격식·진입장벽. BRAND 안티 포지셔닝과 정면 충돌.
- 적합한 제품: 출판·법무·전통문화 (이 스킬의 디폴트 대상은 아님)

> ★ = BRAND 친화 / ◆ = 조건부 / △ = 신중히 / ✗ = 거의 회피

## 3. 다양성 룰 (Convergent 단계)

최종 5~7개 후보 추릴 때 적용:

1. **최소 4개 카테고리** 이상에서 출력. 한 카테고리에 3개 이상 몰리면 재추림.
2. **★ 카테고리 우선** (1·3·6). ◆ 카테고리는 1~2개 포함. △·✗는 0~1개.
3. **점수만으로 정렬 금지**. 점수 80인 후보 2개가 같은 카테고리면, 75점 다른 카테고리 후보가 우선.

---

## 4. Divergent → Convergent 흐름

```
[Divergent] (20~30개 생성)
  ├── 카테고리 1·3·6 (★): 각 3개씩 = 9개
  ├── 카테고리 2·4·7 (◆): 각 2~3개씩 = 6~9개
  ├── 카테고리 5·8·9 (△): 각 1~2개씩 = 3~6개
  └── 카테고리 10 (✗): 0~1개

[클러스터링]
  ├── 음/뜻이 비슷한 후보 묶기
  └── 각 클러스터 대표 1개만 남기기

[Convergent] (5~7개 선별)
  ├── signals-naming.md 점수 계산
  ├── 자동 폐기 조건 적용
  ├── 카테고리 다양성 룰 적용 (§3)
  └── 점수 + 다양성 + 의외성 종합 → 최종 5~7개
```

## 5. 카테고리 추론 (시드 → 권장 카테고리 가중)

시드를 보고 어떤 카테고리에 더 무게를 둘지 자동 결정:

| 시드 특성 | 카테고리 가중 |
|---|---|
| 일상·기록·습관 | 1(의태어), 3(메타포) |
| 게임성·가벼움·위트 | 6(펀), 7(의인화) |
| 명상·심리·내면 | 3(메타포), 5(시적) |
| 도구·생산성 | 3(메타포), 4(신조어) |
| 글로벌/B2B | 9(외래어) — 단, BRAND 톤 우선 |
| 추상 가치/철학 | 5(시적), 6(펀) |

시드 분석 후 위 표를 참조해 Divergent 단계의 카테고리별 생성량을 조정한다.
