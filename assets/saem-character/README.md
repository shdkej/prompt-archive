## Identity Rules

- 손바닥 크기의 둥글게 닳은 조약돌 형태를 유지한다.
- 몸체는 따뜻한 회베이지색 매트 스톤 질감이다.
- 머리 위쪽에는 부드러운 올리브색 이끼 패치가 있다.
- 작은 새싹은 허용되며, 축적/성장을 보여주는 핵심 표식이다.
- 얼굴은 작고 담담한 점 두 개의 눈만 쓴다.
- 입은 없거나 거의 보이지 않게 둔다.
- 사람형, 로봇형, 동물형으로 바꾸지 않는다.
- 크고 반짝이는 눈, 과장된 표정, 유치한 귀여움은 금지한다.
- 이미지 안에 텍스트, 숫자, 로고, UI 아이콘을 직접 생성하지 않는다.
