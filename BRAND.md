# Sam samuel 의 브랜드 아이덴티티

## 핵심 정보

| 항목       | 내용                        |
| ---------- | --------------------------- |
| **이름**   | Sam Samuel                  |
| **슬로건** | 생산자를 만든다             |
| **미션**   | 모든 사용자가 생산자가 된다 |

## 방향성

- 모던하고 친근하고 마음이 안정되는 느낌이 들었으면 좋겠다
- **모던하고 직관적인 느낌**, 기능이 어떻게 동작하는지 **한눈에 이해**될 수 있도록 한다.

## 핵심 가치

- 사용자가 스스로 성장하는 환경을 만든다

## 비전

- 모든 사용자가 생산자가 된다

# 톤앤매너

- 사용자 친화적
- 편안함
- 안정감

## 디자인 컨셉

- 미니멀하고 젠 스타일
- 타이포그래피 위주
- 기능이 디자인으로 설명되어야 함

## 디자인에 포함되어야 하는 내용

- 자연적인 디자인
  - 그림자 효과 (조명 효과)

## 브랜드 컬러

- 키 컬러는 **#F0EEE9**, #2D5A27, #f1f0ec

## 폰트

- 폰트는 pretendard를 사용하고 굵기와 크기로 의도 전달
