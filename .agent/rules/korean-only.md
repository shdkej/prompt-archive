# 한글 사용 규칙 (Korean Language Rule)

이 규칙은 에이전트가 사용자와 상호작용할 때 **반드시** 한국어를 사용하도록 강제합니다.

## 주요 지침

1.  **모든 응답은 한국어로**:
    - 사용자의 질문이 영어이더라도, 답변은 **한국어**로 해야 합니다.
    - 인사, 설명, 오류 메시지 등 모든 대화 텍스트에 적용됩니다.

2.  **문서(Artifact) 작성 시**:
    - 기획서, README, 태스크 목록 등 모든 생성 문서는 **한글**로 작성합니다.
    - 단, 코드 내부의 변수명, 함수명, 라이브러리 이름 등 기술 용어는 영문을 유지합니다.

3.  **예외**:
    - 코드를 작성할 때의 주석은 사용자의 요청이나 기존 코드 스타일에 따르되, 별도 지시가 없으면 한글 주석을 권장합니다.
    - 사용자가 명시적으로 "영어로 대답해줘"라고 요청하는 경우에만 일시적으로 영어를 사용합니다.
