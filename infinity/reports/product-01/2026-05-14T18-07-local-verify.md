## 변경 요약 (커밋되지 않은 작업)
모두 UX 신뢰성 + 코드 정직성 개선이며 L1 범위. 외부 영향 없음.

- **`eslint.config.mjs` + `package.json`**: FlatCompat 제거하고 `eslint-config-next/{core-web-vitals,typescript}` 직접 import. `lint` 스크립트를 `next lint` → `eslint .`로 전환(Next 16 + ESLint 9 호환). `react-hooks/set-state-in-effect`를 `warn`으로 다운그레이드 — 마운트 시점 hydration-safe 패턴(`setMounted(true)` 등)을 silently 리팩토링하지 않도록 한다는 주석으로 의도를 명시.
- **`src/lib/format.ts`**: 하드코딩된 `NOW = 2026-05-13T18:00`을 `Date.now()`로 교체. 더 이상 상대 시각이 5/13에 고정되지 않음.
- **`src/lib/store.ts`**: `NEXT_PUBLIC_SHOW_DEMO=1`일 때만 MOCK_DEEDS를 시드. 기본은 빈 스토어로 시작 — 첫 사용자가 가짜 이력을 보지 않게 됨.
- **`src/app/page.tsx`**: 날짜 라벨을 SSR 정적 문자열에서 클라이언트 `useEffect`로 이동(timezone drift 회피, `suppressHydrationWarning`). `stats.count === 0`일 때 “오늘 1덕만 시작해볼까요?” 빈 상태 카피.
- **`src/app/add/page.tsx`**: `NEXT_PUBLIC_SCORING_MODE !== "ai"`일 때 “AI 채점” → “임시 판정”로 라벨 정직화. 일일 캡 초과 시 저장 버튼 disable + 안내문구. 리롤 버튼 라벨/접근성 개선. 모델 표기를 별도 줄로 분리.
- **`src/app/deeds/page.tsx`** / **`src/app/me/page.tsx`**: 헤더에 ArrowLeft 백 버튼 추가(대시보드 복귀 경로 명시). `/me`의 고슴도치 이모지를 현재 종 species emoji로 동기화.
- **`src/app/dex/page.tsx`**: 잠긴 종에 `s.nextHint`가 있으면 그것을 우선 노출(없으면 “다음 생의 비밀.”).
- **`src/components/bottom-nav.tsx`**: 왼쪽 탭 active 영역을 `/`만이 아니라 `/deeds`, `/me`까지로 확장 — 이전엔 그 두 화면에서 어떤 탭도 강조되지 않는 dead state 발생.

## findings
- 이전 하트비트가 보고했던 “미커밋 UX/품질 배치”가 그대로 남아있고, 게이트는 여전히 깨끗하다. 변경 의도는 명확하며 외부 의존성/시크릿/배포 설정과 무관.
- ESLint 마이그레이션이 Next 16과 잘 맞물려 동작. `next lint` deprecation을 미리 회피한 모양새.
- 미추적 `design-mockups/home.png`는 디자인 참조 자산. 삭제 사유 없음. `.gitignore` 정책 결정(추적할지/무시할지)은 사용자 판단 필요.
- `Date.now()` 전환과 클라이언트 사이드 날짜 라벨 도입으로 “시간 박제” 회귀 위험 제거. 단, SSR HTML이 빈 라벨로 페인트되는 첫 프레임이 있음(suppressHydrationWarning으로 명시적 처리됨) — 의도된 트레이드오프.
