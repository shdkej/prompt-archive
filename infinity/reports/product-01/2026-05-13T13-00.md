### route.ts 구현 스케치

`infinity/artifacts/product-01/ai-scoring-prompt.md` 에 이미 작성된 초안을 기반으로 구현한다.

핵심 포인트:
- `@anthropic-ai/sdk` 사용 (이미 패키지에 있는지 확인 필요, 없으면 `pnpm add @anthropic-ai/sdk`)
- `zod` 사용 (이미 있는지 확인 필요, 없으면 `pnpm add zod`)
- 환경변수 `ANTHROPIC_API_KEY` — `.env.local` (`.gitignore`에 포함되어 있는지 확인)
- 이미지: base64 + mimeType (클라이언트에서 `FileReader` 로 변환 후 전달)
- 메모: 선택적 텍스트
- 톤: `soft` | `casual` (기본 `soft`)

```typescript
// src/app/api/score/route.ts
import Anthropic from "@anthropic-ai/sdk";
import { z } from "zod";
import { NextRequest } from "next/server";

const ScoreSchema = z.object({
  score: z.number().int().min(0).max(10),
  comment: z.string().max(80),
  tags: z.array(z.string()).max(2),
});

const SYSTEM_PROMPT = `...`; // ai-scoring-prompt.md 내용 그대로

export async function POST(req: NextRequest) {
  const { imageBase64, mimeType, memo, toneMode = "soft" } = await req.json();

  const client = new Anthropic();
  const systemPrompt = SYSTEM_PROMPT.replace("{{toneMode}}", toneMode);

  const response = await client.messages.create({
    model: "claude-sonnet-4-6",
    max_tokens: 256,
    system: systemPrompt,
    messages: [{
      role: "user",
      content: [
        { type: "image", source: { type: "base64", media_type: mimeType, data: imageBase64 } },
        ...(memo ? [{ type: "text" as const, text: memo }] : []),
      ],
    }],
  });

  const raw = response.content[0].type === "text" ? response.content[0].text : "";
  const parsed = ScoreSchema.parse(JSON.parse(raw));
  return Response.json(parsed);
}
```

변경 후 흐름:
1. 이미지 선택 → blob URL 미리보기 + **FileReader로 base64 변환**
2. 메모 입력
3. "채점하기" 버튼 → `fetch("/api/score", { method: "POST", body: JSON.stringify({ imageBase64, mimeType, memo, toneMode }) })`
4. 로딩 상태 표시 (스피너 or 버튼 비활성화)
