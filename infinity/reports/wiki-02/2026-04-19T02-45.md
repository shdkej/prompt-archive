## Lessons (OPERATING_LESSONS 반영 제안)

- Heartbeat Agent가 "환경 제약"으로 blocked 처리하기 전에 **대안 경로 최소 N개**를 탐색하도록 프로토콜 보강 필요 (예: MCP 실패 시 gh CLI, git+PAT, 웹 UI 순 확인)
- "웹 UI 필수" 같은 기술 제약 주장은 근거 링크나 실제 시도 결과가 없으면 신뢰하지 말고 재검증
- 반복 blocked 리포트가 N회 이상이면 자동으로 승인 요청을 격상하거나 대안 경로 제안을 사용자에게 요약 알림
