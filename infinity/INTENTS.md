## Inbox

- [리서치/스킬화] 바이럴 키워드 추천 스킬 분석자료 만들기
  - registered_at: 2026-05-05T07:44:23Z
  - requester: user
  - goal: 사용자가 글을 쓸 때 적절히 바이럴될 만한 키워드를 찾아주는 AgentSkill을 만들 수 있도록, 기존 키워드 추천/트렌드 분석 시스템을 조사·비교하고 스킬 설계 자료로 정리한다.
  - scope:
    - 기존 키워드 추천 시스템/도구/워크플로우 사례 분석
    - 바이럴 가능성 판단에 쓰이는 신호 정리: 검색량, 상승 추세, 경쟁도, 플랫폼별 맥락, 제목/본문 적합도, 롱테일 여부, 타깃 독자 반응 가능성 등
    - OpenClaw AgentSkill로 구현할 때 필요한 입력/출력 포맷, 도구 흐름, 데이터 소스, 캐시/비용 절약 전략 제안
    - 글쓰기 지원용으로 실제 사용 가능한 추천 결과 템플릿 제안
  - expected_output: 분석자료 문서와 스킬 설계 초안. 가능하면 후속으로 Skill 생성에 바로 쓸 수 있는 구조까지 포함.
