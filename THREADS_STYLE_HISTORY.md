# THREADS_STYLE_HISTORY

Threads 문체 컨펌 히스토리 로그.

## 규칙
- 항목마다 날짜, 상태(승인/수정요청), 피드백 요약, 샘플 문장 기록
- 동일 주제라도 문체 피드백이 바뀌면 새 항목 추가
- 다음 초안 작성 전 최근 5개 항목 우선 참조
